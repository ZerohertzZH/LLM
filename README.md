# LLM
我想用AI来做一些有趣的事情，比如写小说、编程、翻译等等。这个项目就是为了实现这些功能而创建的。
## 功能
- 写小说：输入一个主题，AI会生成一篇小说。
- 编程：
>*  ①输入一个规则，AI会把他记录到自己的知识库中。
>*  ②输入设计文档，AI会帮你做编程问题。
>*  ③输入设计文档和代码，AI会给你做测试，并且帮你找出bug，生成测试报告。
- 翻译：输入一段文本，AI会翻译成指定的语言。
## 技术
这个项目使用了以下技术：
- Python：主要编程语言。
- OpenAI API：用于生成文本和翻译。
- Flask：用于创建Web应用。
- SQLite：用于存储规则和设计文档。
## 使用方法
1. 克隆这个仓库到你的本地。
2. 安装依赖：
```
pip install -r requirements.txt
```
3. 运行应用：
```
python app.py
```
4. 打开浏览器，访问`http://localhost:5000`，你就可以使用这些功能了。
